# Brand Assets

Add your Evolution Racing brand assets to this folder.

## Supported Formats
- PNG
- JPG/JPEG
- SVG
- WEBP

## Recommended Assets
- First Gear logo variants
- Evolution wordmark
- Brand patterns/textures
- Social media templates
- Race graphics
- Trainer/horse photos

## Naming Convention
Use descriptive names:
- `logo-first-gear-black.png`
- `logo-evolution-gold.svg`
- `pattern-gradient-black-gold.png`
- `template-race-update.jpg`

Assets will automatically appear in the dashboard selector once added.
